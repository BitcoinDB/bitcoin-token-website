This page contains instructions for how to install BitcoinWallet, BitcoinDb, and BitcoinToken in a browser and in node.

BitcoinWallet

The easiest way to get started is to use BitcoinWallet in node.

Node

You need node.js and npm installed. Then create a file index.js

          const Bitcoin = require('bitcointoken')
          const wallet = new Bitcoin.Wallet()
          
          console.log(`
          \tprivateKey: ${wallet.getPrivateKey()}
          \taddress: ${wallet.getAddress()}
          `)

Type node --experimental-repl-await index.js into a terminal window. If everything worked, you should see output similar to this:

          privateKey: da31ca9a791c6214069b5e87f8080ba69d94336dee54f0421c4019d9e41ddca8
          address: mwZd1bgRYhxY4JLyx1sdEGBYFZnXVDvgmp

Browser

Install Parcel using npm.

          npm install -g parcel-bundler

Initialize an empty project

          npm init -y

Create a file called index.js with the following content

          const Bitcoin = require('bitcointoken')
          const wallet = new Bitcoin.Wallet()

          document.body.innerHTML = 
           `<h1>Your Wallet</h1>
            privateKey: ${wallet.getPrivateKey()}<br>
            address: ${wallet.getAddress()}`

Then create a file called index.html

          <!doctype html>

          <html lang='en'>
            <body>
              <script src="./index.js"></script>
            </body>
          </html>

Start a development server using

          parcel index.html

You can see the rendered website at http://localhost:1234/
