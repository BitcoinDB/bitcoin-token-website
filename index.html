Launch a Token in a few lines of Javascript

BitcoinToken allows you to build your own token on top of Bitcoin Cash. It consists of a minimal standard called the Simple Token Standard as well as a reference implementation. The Simple Token Standard defines a way in which the stake holders of a token can agree on what consitutes a valid token. The definition of what constitutes a valid token is provided by the developer of the token as a Javascript function.
