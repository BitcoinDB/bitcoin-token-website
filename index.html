Build smart contract powered apps

It's easier than you think

BitcoinToken is a suite of tools to integrate payments and smart contracts into your web app. We currently offer three products: a client-side wallet, a database system for storing data on the blockchain, and a token solution. Our smart contracts are compiled into Bitcoin Cash, which makes it possible to operate them at a cost that is orders of magnitude below the competition. The tools are easy to use, no prior blockchain knowledge is required.

We are in prototype stage and plan to launch towards the end of the year.

BitcoinWallet - a wallet for your money and your data

BitcoinWallet is a Bitcoin Cash wallet that works out of the box. Just install the npm package and you are ready to store, send, and receive Bitcoin Cash. It works in the browser and in node which makes it possible to write non-custodial applications.

BitcoinWallet not only stores the keys to hold your Bitcoin Cash, it also stores keys that allow you to store your data on the blockchain.

BitcoinDb - store data forever at a one-time cost

BitcoinDb allows you to store, read, and update JSON objects on the Bitcoin Cash blockchain. It is always online, available worldwide, and extremely censorship resistant. It syncs across the globe within seconds making it possible to write real-time applications on top of it.

BitcoinDb stores every JSON object in an atomic unit with an amount of Bitcoin Cash. All data stored in BitcoinDb is globally visible, however, updates can be restricted using the same security apparatus that protects your Bitcoin.

BitcoinToken - issue tokens and smart contracts

BitcoinToken allows you to build Bitcoin and smart contract enabled apps without prior knowledge of Bitcoin. For example, you can issue your own token in a few lines of Javascript.

We currently support ERC20-style fungible tokens. Support for non-fungible tokens will be added soon. Eventually, you will be able to program your own smart contracts using BitcoinToken.
