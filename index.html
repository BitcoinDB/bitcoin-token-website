Bitcoin Token

Build tokens in Bitcoin Cash

BitcoinToken is a tool for developers to build tokens and other smart contracts on top of Bitcoin Cash. It grants the programmer very low level access, think of it as writing smart contracts in assembly code. You can also use a pre-defined library and launch a token in a few lines of Javascript. We are in prototype stage and plan to launch towards the end of the year.

Simple

Launching a token is as simple as running a few lines of javascript code. 

const bitcoinToken = new BitcoinToken()
await bitcoinToken.init('./erc20.js')
bitcoinToken.name = 'MyToken'
bitcoinToken.supply = 21000000
await bitcoinToken.create()
await bitcoinToken.send(500, myFriendsPublicKey)

Expressive

BitcoinToken relies on the well understood and battle tested ideas behind colored coins. However it greatly expands the expressiveness by allowing the issuer/developer of the token to define what a valid token transfer transaction is. For example, to define an ERC 20 token, the programmer will need to provide the following code to launch a token. . 

async function isValid(tx) {
  const sumInputs = tx.inputs.reduce((a, b) => a.data + b.data, 0)
  const sumOutputs = tx.outputs.reduce((a, b) => a.data + b.data, 0)
  return sumInputs === sumOutputs
}

Scalable

BitcoinToken is as scalable as the underlying blockchain. Issuing a new token or sending tokens to another user can be done within seconds. Receiving tokens is usually the bottleneck for token solutions on top of Bitcoin. However our solution guaranties lightning fast token validation with minimal storage overhead.
