BitcoinToken is a suite of tools that make it easy to build smart contracts on top of Bitcoin Cash. All tools can be used without prior knowledge of Bitcoin.

BitcoinToken

Create tokens and smart contracts on Bitcoin Cash

BitcoinDb

Store, update, and retrieve json data on the Bitcoin Cash blockchain.

BitcoinWallet

Receive, hold, and send Bitcoin Cash.

The BitcoinToken suite consists of three classes, all of which are exposed at the root level. 

            const Bitcoin = require('./bitcoin-token')

            const BitcoinWallet = Bitcoin.Wallet
            const BitcoinDb = Bitcoin.Db
            const BitcoinToken = Bitcoin.Token

BitcoinWallet

BitcoinWallet is a Bitcoin wallet that works out of the box. It allows users to store, send, and receive Bitcoin Cash.

Initialization

There are two ways to initialize a BitcoinWallet. To create a wallet and initialize it with a fresh random master private key, just call the constructor. 

           const wallet = new BitcoinWallet()

To initialize the wallet from an existing private key call fromHdPrivateKey 

            const masterPrivateKey = 'tprv8ZgxMBicQKsPefX1...'
            const wallet = BitcoinWallet.fromHdPrivateKey(masterPrivateKey)

Receiving Bitcoin Cash

You wallet comes with over 2 billion accounts, numbered from 0 to 2.147.483.648. You can access these accounts by executing the following commands. The default account is 0. 

            const accountNumber = <some number between 0 and 2.147.483.648>
            const privateKey = wallet.getPrivateKey(accountNumber)
            const publicKey = wallet.getPublicKey(accountNumber)
            const address = wallet.getAddress(accountNumber)

To fund the wallet get a deposit address and fund through a testnet faucet (eg https://coinfaucet.eu/en/bch-testnet/).

Sending Bitcoin Cash

After you have funded your wallet, you can send Bitcoin Cash to another address as follows. 

            const amount = <number of satoshi to be sent>
            const address = <some Bitcoin Cash address>
            const res = wallet.send(amount, address)

By default, the wallet will send funds from account number 0. If you want to send from another account, plug in that accounts number as the third parameter 

            const accountNumber = <an account number between 0 and 2.147.483.648>
            const res = wallet.send(amount, address, accountNumber)

Checking the balance

You can check the balance of an account by calling getBalance with the account number. Like everywhere else the account number can be ommited and defaults to 0. 

            const balance = wallet.getBalance(accountNumber)

BitcoinDb

BitcoinBd lets you store, read, and update json files on the blockchain. Every json file stored on the blockchain has an owner and only the owner can update that piece of json. We are currently only supporting json objects with string values of less than 350 charachters. In the future we will add support for arbitrary json data.

Initialization

There are three ways to initialize a BitcoinWallet: To create a wallet and initialize it with a fresh random master private key, just call the constructor. To initialize from an existing private key call fromHdPrivateKey. The third option is to pass in a bitcoinDb object 

            const bitcoinDb1 = new BitcoinDb()

            const masterPrivateKey = 'tprv8ZgxMBicQKsPefX1...'
            const bitcoinDb2 = BitcoinDb.fromHdPrivateKey(masterPrivateKey)

            const bitcoinWallet = new BitcoinWallet()
            const bitcoinDb3 = new BitcoinDb(bitcoinWallet)

Storing data

To store some data in json format create and object containing that data as a value under the key data. 

            const data = { a: 'aaa', b: 'bbb' }
            await bitcoinDb1.put({ data })

Anyone can read the data that is stored in BitcoinDB, however only the owner can update the data. By default the current user will be the owner of newly created data. To designate another user, just pass in that users public key 

            const data = { a: 'aaa', b: 'bbb' }
            const owners = [ bitcoinDb2.getPublicKey() ]
            await bitcoinDb1.put({ data, owners })

Reading data

The value returned from bitcoinDb.put is an object with one key txId and one key outputNumber. This id can be used to read the json object from the blockchain. You can pass that identifier into bitcoinDb.get to retrieve the json object. 

            const id = await bitcoinDb.put({ data: { c: 'ccc' } })
            const data = await bitcoinDb.get(id)

data is an object with three keys: data contains the initially stored data, publicKeys (to be renamed owners) is the list of the public keys of the owners, and amount contains the number of satoshi contained in the output that stores the data.
