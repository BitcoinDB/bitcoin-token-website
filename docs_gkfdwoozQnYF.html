Documentation

BitcoinToken is a suite of tools that make it simple to build smart contracts on top of Bitcoin Cash. It consist of three products:

  • BitcoinWallet - a wallet that works out of the box, no configuration required
  • BitcoinDb - a tool to reade, write, and update data on the Bitcoin Cash blockchain
  • BitcoinToken - a took that makes it east to create tokens and smart contracts

BitcoinWallet

BitcoinWallet is based on hierarchical deterministic wallets as specified in BIP32.

Initialization

A wallet stores one master private key and can generate a large number of private-public keypairs from this private key. To create a wallet and initialize it with a fresh random master private key run the following 

const BitcoinTools = require('./bitcoin-tools')
const Wallet = BitcoinTools.Wallet
const wallet = new Wallet()

You wallet comes with a large number of bitcoin accounts, numbered from 0 to some large number. You can access these by executing the following commands. 

const index = <some number between 0 and some large number>
const privateKey = wallet.getPrivateKey(index)
const publicKey = wallet.getPublicKey(index)
const address = wallet.getAddress(index)

To fund the wallet get a deposit address and fund through a testnet faucet (eg https://coinfaucet.eu/en/bch-testnet/). Alternatively, you can initialize a wallet object from an private key in wif format run the following 

const masterPrivateKeyString = 'tprv8ZgxMBicQKsPdrjKM2Ap4MDw5PawXdavaJJ23e5eDHjYupUc6NUGLvS3Tp4e9dM36PBP7zpi48PLmPYgAt9qNX1BDxnEfSv7ccFk91PR8Xf'
const walletAlice = Wallet.fromHdPrivateKeyString(masterPrivateKeyString)

Sending bitcoin

To send Bitcoin Cash to another address run the following 

const amount = <number of satoshi to be sent>
const address = <some Bitcoin Cash address>
const res = wallet.send(amount, address)

By default, the wallet will send funds from account number 0. If you want to send from another account, plug in that accounts number as the thrid parameter 

const accountNumber = <an account number between 0 and a large number>
const res = wallet.send(amount, address, accountNumber)
